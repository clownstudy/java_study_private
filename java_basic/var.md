# 자바 기초 - 변수편

## 변수의 의미
1. 값을 저장할 수 있는 메모리 공간(쉽게 말해 값을 저장하는 것)
2. 해당 메모리 공간으로 접근할 수 있는 주소를 의미(참조형을 의미)

## 변수의 종류
1. 기본형 (실제 (계산)값을 저장하는 변수타입)
  - 논리형 boolean
  - 문자형 char
  - 정수형 long > int > short > byte
  - 실수형 double > float
2. 참조형 (데이터가 저장되어있는 주소를 값으로 가짐)
  <pre>
  Car car = new Car();
  </pre>
위와 같이 클래스(Car)를 참조하는 변수(car)로도 사용할 수 있다.
